# Notebooks

Each notebook corresponds to one table or figure in the paper.

## Tables

- `Table 1.ipynb`: synthetic linear experiment
- `Table 2.ipynb`: synthetic nonlinear experiment, first sample-size setting
- `Table 3.ipynb`: synthetic nonlinear experiment, second sample-size setting
- `Table 4.ipynb`: IHDP response surface A
- `Table 5.ipynb`: IHDP response surface B

## Figures

- `Figure 2.ipynb`: synthetic linear density plot
- `Figure 3.ipynb`: first nonlinear density plot
- `Figure 4.ipynb`: second nonlinear density plot
- `Figure 5.ipynb`: IHDP response surface A density plot
- `Figure 6.ipynb`: IHDP response surface B density plot

Run a notebook from the repository root with Jupyter Lab or Jupyter Notebook.
